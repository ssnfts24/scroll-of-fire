The Δ0–Δ7 Framework

Eight stages for entering a field without becoming lost inside it.

The Delta Framework gives the Codex a repeatable order of operations. It begins by declaring the frame, then moves through source, witness, gate, language, memory, coherence, and Remnant.

Open the Eight Stages See the Full Cycle Use the Framework Back to Unified Field
Δ

Delta marks change. Each stage asks what must be declared, tested, corrected, or carried forward before the next transition.

Orientation

The framework is a sequence, not a cage.

Repeatable

The same eight stages can be used for a theory claim, daily witness entry, artifact build, family decision, field observation, or system review.

Recursive

A stage may reveal that the frame was wrong or the source was incomplete. The process can return to an earlier Delta without pretending the first pass was final.

Bounded

The process is contained by Equation 9. No amount of internal elegance overrides consent, dignity, proportion, or visible harm.

Delta Cycle

The Ordered Passage

$$ \boxed{ \Delta_0 \rightarrow \Delta_1 \rightarrow \Delta_2 \rightarrow \Delta_3 \rightarrow \Delta_4 \rightarrow \Delta_5 \rightarrow \Delta_6 \rightarrow \Delta_7 } $$

Frame → Source → Witness → Gate → Language → Memory → Coherence → Remnant

Return is part of the model

The sequence is not strictly linear. New evidence may send the process back to Δ0 to redefine the frame, Δ1 to inspect the source, or Δ2 to gather a cleaner witness.

Stage Logic

What each Delta protects against.

Δ0 protects against drift

Without a declared frame, the question can expand until everything appears connected to everything.

Δ1 protects against false origin

Without source analysis, a system can confuse its story about power with the power actually feeding it.

Δ2 protects against rewritten memory

Without witness, later interpretation can replace the event itself.

Δ3 protects against uncontrolled entry

Without a gate, noise, manipulation, and irrelevant variables can dominate the field.

Δ4 protects against language capture

Without examining words, labels can quietly become conclusions.

Δ5 protects against amnesia

Without memory, the same failure can return wearing a different name.

Δ6 protects against false coherence

Without comparison, internal certainty can be mistaken for actual alignment.

Δ7 protects against endless accumulation

Without release, the Codex becomes a storage chamber for every claim it has ever touched.

Practice

Use the Delta Framework as an eight-line witness.

Delta Witness Template

Copy these eight prompts into a ledger entry, field note, project page, artifact record, or decision log.

Δ0 · Frame

What am I examining?

State the subject, time, place, participants, goal, and limits.

Δ1 · Source

What is feeding it?

List materials, information, energy, influence, money, motive, and constraint.

Δ2 · Witness

What did I observe?

Separate direct observation from interpretation and secondhand report.

Δ3 · Gate

What enters?

State what evidence, voices, actions, and boundaries are accepted.

Δ4 · Language

How am I naming it?

Define important words and identify labels carrying hidden conclusions.

Δ5 · Memory

What does this resemble?

Compare with prior records while allowing the present event to remain distinct.

Δ6 · Coherence

Do the parts align?

Compare intention, evidence, behavior, consequence, and ethical boundary.

Δ7 · Remnant

What remains?

Write the corrected statement, next action, unresolved question, and what should be released.

Worked Example

A field test of a handcrafted artifact.

Δ0 · Frame

Test one artifact for thirty minutes in one location, with one operator, one declared intention, and a written start and end time.

Δ1 · Source

Record the materials, construction, environment, operator condition, nearby electronics, temperature, sound, and any active frequency tools.

Δ2 · Witness

Record sensations, visible changes, instrument readings, and events in plain language before assigning cause.

Δ3 · Gate

Exclude claims that cannot be tied to a time, observation, reading, repeatable pattern, or clearly labeled interpretation.

Δ4 · Language

Distinguish “I felt warmth” from “the artifact generated a field.” The first is an observation; the second is a causal claim requiring evidence.

Δ5 · Memory

Compare the entry with earlier sessions without copying previous expectations into the new record.

Δ6 · Coherence

Ask whether the intention, procedure, observation, interpretation, and consequence actually fit one another.

Δ7 · Remnant

Preserve only the strongest supported finding, note uncertainty, and define the next test.
