Start Here

How to use this site in 3 minutes

  1. Run the 60-second tool above. One line. One slider. Save it.
  2. Skim the Guide. It’s a plain-language overview of the Codex and how each page fits.
  3. When curious, open Theory. The canonical equations (0–17) are there with notes + references.
  4. When practicing, use Teach/Manifest. Voice protocol, A/B carriers, ethics, and group runs.
  5. Record your witnesses in the Ledger. It’s local-first (browser storage) with export.

Tip: headphones help. Keep phrases kind, precise, and present-tense. Stop if anything feels off-balance.

Practice flow

Daily loop (5–10 min)

  • Pick a carrier (432 / 528 / 369).
  • Speak one line for ~60s.
  • Rate coherence (0–5); note any sensations / images / words.
  • Save to Ledger; add a tag (e.g., morning, 432).
  • Optionally A/B tomorrow (swap carrier or phrase weight).

See A/B carriers and phrase weights.

Safety & ethics

Do no harm, seek coherence

  • Stop a run if you feel dizzy, pressured, or emotionally flooded.
  • Prefer truthful, non-coercive phrases (no control language).
  • Group work? Read the Ethics page and apply the stop-rules.

The Codex is a learning framework, not medical advice.
