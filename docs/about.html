System Definitions

What Each System in the Codex Actually Does

The following definitions describe the actual function of each named system within Codex of Reality, using language present in the official site content.

☾ 13 Moons The Remnant 13 Moons calendar is a daily rhythm system that gives each day a mirror: moon phase, symbolic tone, witness questions, and a closing seal. It is used as a reflective practice for attention, pattern, and intentional action — not as a system of belief or prediction. Open 13 Moons
☲ Witness Ledger The Witness Ledger is a daily record-keeping practice that turns lived experience into trackable signal. It captures date, moon position, repeated signals, body awareness, pressure, and a clean next action. It makes observation durable rather than letting it vanish. Open Ledger
∿ Frequency Governance Frequency Governance is a carrier tone system built around specific frequencies (432 Hz, 528 Hz, 369 Hz, 144 Hz, 963 Hz) used for resonance and coherence practice. It includes structured listening, a visual field interface, and a voice gate for humming and stability measurement.

◎ Genesis Oracle The Genesis Oracle is a symbolic mirror tool for language, questions, timing, and reflection. It generates symbol-based prompts to support observation, slow deliberate thinking, and directional clarity. It does not predict outcomes or claim prophetic authority. Open Oracle
Ξ T-7 Resonant Engine The T-7 (T-7 Resonant Engine) is an advanced carrier-mixing console for phase, coherence, memory, resonance practice, field visualization, and shareable session states. It is a functional audio and visual interface within the Frequency Governance system.

⚒ Artifact Registry The Artifact Registry documents handcrafted copper artifacts, living technology pieces, tools, and physical creations made by Aaron Paul Laird. Each entry includes valuation, materials, build notes, and archive record. Open Registry
⛺ Covenant Caravan The Covenant Caravan is a framework for resilience, craft, family culture, movement, portable economy, and road systems. It documents the practical infrastructure of a mobile or off-grid life built around real skills and shared covenant. Open Caravan
∑ Theory The Theory section contains the mathematical and symbolic framework underlying the Codex: the Master Equation, Canon 0–17, operators, unified field structure, memory, ethics, notation, and frequency governance theory. All equations are defined within the framework and documented on this site.
