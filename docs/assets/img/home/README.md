# home/

**Public path root:** `assets/img/home/`
**Purpose:** Homepage-only artwork for the Codex of Reality.

## Subfolders

| Folder | Status | Notes |
|--------|--------|-------|
| `hero/` | planned | Disabled hero replacement artwork only |
| `gates/` | mixed | One active social image, other staged homepage gate images |
| `sections/` | planned | Future homepage section illustrations |
| `backgrounds/` | planned | Disabled background textures only |
| `posters/` | planned | Future poster or background treatments |
| `social/` | planned | Homepage-specific OG image not yet enabled |

## Path rules

- Keep homepage-specific files inside the appropriate subfolder.
- Use lowercase kebab-case filenames.
- Do not enable staged assets in live HTML or CSS until they are explicitly verified.
- Reusable assets belong under `assets/img/shared/`, not under `assets/img/home/`.
