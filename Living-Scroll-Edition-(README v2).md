🔍 Deep Link / URL Parameters

Genesis Oracle and the mini bar accept a parameterized link:

?name= — prefill name

&dob= or &date= — YYYY-MM-DD

&t= or &time= — HH:MM

&tz= — IANA TZ (e.g., America/New_York)

&leap=1|0 — include Leap Day in 13×28

&doot=1|0 — include Day Out of Time

&glow=1|0 — enable Remnant glow for key sums

&carrier=auto|432|528|369|144|963

&t2=laws|flame|name — open Tier II tab

&name2= — Tier II Name override
